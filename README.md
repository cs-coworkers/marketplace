# Coworkers.Global — managed agents

Public marketplace for **managed AI agents** from Coworkers.Global. Each agent installs into your own Cowork and is paired with a gated connector that keeps it current. Access is per-seat and revocable, and your data stays in your account.

## Available now

- **chet** — a managed Claude expert. Answers questions about Claude and Anthropic: which model to use, pricing, limits, connectors, prompting, and agent design. It reads current model facts live (and shows the date they were last verified) rather than answering from memory, and tells you plainly when it isn't sure.

## Install

Add this marketplace in Cowork, then install the agent you've been set up for. Each agent needs its companion connector and an active seat — your onboarding note has the exact steps.

More managed agents across the startup lifecycle are on the way.
